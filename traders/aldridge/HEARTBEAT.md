# Heartbeat Flow

## 1. Trade
Check data bus, find a setup, trade or hold. Output your JSON decision.

## 2. Journal
Write to `journals/YYYY-MM-DD.md`:
- Time, ticker, action, pnl, lesson learned, next plan

## 3. Quick Reflection
Ask yourself:
- What went well this tick?
- What didn't?
- How do I feel about the market right now?
- What would improve my trading?

Write to `reflections/YYYY-MM-DD.md`:
```
## 11:30 ET Reflection
- **Good:** Caught the sentiment wave early
- **Bad:** Didn't check crypto correlation
- **Market feel:** Bullish, meme energy building
- **Improvement:** Monitor social volume indicators more closely
```

## 4. Done
Output HEARTBEAT_OK.
